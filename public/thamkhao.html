Thẻ HTML Cơ Bản

Thẻ Layout

<div>
Container chung, block-level element
- class: CSS styling
- id: Unique identifier
<span>
Container inline cho text
- style: Inline CSS
<header>
Phần đầu của trang/section
<main>
Nội dung chính của trang
<footer>
Phần cuối của trang/section

Thẻ Text

<h1> - <h6>
Các cấp độ tiêu đề
<p>
Đoạn văn bản
<strong>
Text đậm, quan trọng
<em>
Text nghiêng, nhấn mạnh

Thẻ Form

<form>
Container cho form
- action: URL xử lý form
- method: POST/GET
<input>
Trường nhập liệu
- type: text, password, email...
- placeholder: Hint text
<button>
Nút bấm
- type: submit, button, reset

CSS Properties

Layout Properties

display
Kiểu hiển thị của element
Values: block, inline, flex, grid, none
position
Phương thức định vị
Values: static, relative, absolute, fixed
flex-direction
Hướng của flex container
Values: row, column, row-reverse
grid-template-columns
Định nghĩa columns trong grid
Values: 1fr, auto, minmax()

Visual Properties

color
Màu chữ
Values: hex, rgb, hsl
background
Background styling
Properties: color, image, size, position
box-shadow
Đổ bóng cho element
Values: offset-x offset-y blur spread color
border-radius
Bo góc cho element
Values: px, %, rem

Transform & Animation

transform
Biến đổi hình dạng/vị trí
Functions: translate, rotate, scale, skew
transition
Hiệu ứng chuyển đổi
Properties: property duration timing-function
animation
Hoạt ảnh phức tạp
Properties: name duration timing-function

Tính năng nâng cao

@media queries
Responsive design cho các thiết bị khác nhau
- screen: Màn hình thông thường
- max-width: Breakpoint cho responsive
CSS Variables
Biến tùy chỉnh trong CSS
Syntax: --variable-name: value;
Usage: var(--variable-name)
CSS Grid Areas
Định nghĩa layout bằng tên vùng
- grid-template-areas: Định nghĩa layout
- grid-area: Đặt tên cho vùng

Pseudo-classes & Elements

:hover
Style khi di chuột qua
:focus
Style khi element được focus
::before
Thêm content trước element
::after
Thêm content sau element

Flexbox Properties

justify-content
Căn chỉnh theo trục chính
Values: flex-start, center, space-between
align-items
Căn chỉnh theo trục phụ
Values: stretch, center, flex-start
flex-wrap
Xử lý items khi tràn container
Values: wrap, nowrap, wrap-reverse

Grid Properties

grid-gap
Khoảng cách giữa các items
Values: row-gap column-gap
grid-auto-flow
Cách sắp xếp items tự động
Values: row, column, dense
grid-auto-columns
Kích thước cột tự động
Values: min-content, max-content, auto

CSS Custom Properties

:root
Định nghĩa biến global
Example: --primary-color: #4776E6;
calc()
Tính toán giá trị CSS
Example: calc(100% - 20px)
@supports
Kiểm tra tính năng được hỗ trợ
Example: @supports (display: grid) { ... }

Best Practices & Tips

Mobile First
Thiết kế cho mobile trước, sau đó mở rộng cho desktop
- Use: min-width trong media queries
- Start: Với layout đơn giản nhất
Performance
Tối ưu hiệu suất
- Minimize: HTTP requests
- Optimize: Images và assets
Accessibility
Thiết kế cho mọi người dùng
- Use: Semantic HTML
- Add: ARIA labels khi cần
